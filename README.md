# ESP32 DHT11 Monitor

Система мониторинга температуры и влажности в реальном времени.

## Как работает

## Стек технологий
- **ESP32-C3 SuperMini** — микроконтроллер
- **DHT11** — датчик температуры и влажности
- **C++ / Arduino** — прошивка ESP32
- **Python Flask** — веб-сервер
- **SQLite** — база данных
- **Chart.js** — графики в браузере

## Подключение DHT11
| DHT11 | ESP32-C3 |
|-------|----------|
| +     | 3.3V     |
| OUT   | GPIO4    |
| -     | GND      |

## Запуск
1. Установить зависимости:
pip install flask

2. Запустить сервер:
cd server
python server.py

3. Открыть браузер:
http://localhost:5000
